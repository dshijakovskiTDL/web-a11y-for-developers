Welcome!

Hey there! I'm Daniel and I'm so happy you joined this course!

I am a Senior Web Developer and Course Instructor at TestDevLab, working with technologies like React, Vue and Angular, all the way to plain old HTML/CSS/JS.

Web Accessibility is a crucial part of what we do as Web Developers, so I invite you to join me in discovering all of its quirks and techniques needed to make a Web page accessible.

Course information

This course comes with a bunch of materials and resources you can learn from!

This is of course the Demo website we will be using as a guide throughout our lessons.
You will also get access to all the slides and additional materials that contain all the information you will need to learn the topics covered in this course on your own time.

Finally, there is the GitHub repo of this website which we'll use to do some practical work.
Before we start, I recommend you clone the repo and have it open in your favorite IDE.

Topics

This course will take place over 4 days each covering a specific topic of Web Accessibility:

  1. Introduction - What is Web Accessibility and common pitfalls during development.
  2. Components - Building common UI components with accessibility in mind.
  3. Accessible Forms - The techniques and accessibility principles of forms on the Web.
  4. Best Practices - Tips & Tricks. Best Practices & Workflows for any development team.
